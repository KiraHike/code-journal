  • CSS Cascade

    The four components of the cascade are: Source Order, Inheritance, Specificity and !Important. The three selector types in order of increasing specificity are: element type, class, id.

  • DOM Creation

    To insert a new element follow these steps: 1. Create an element node 2. Get the parent element to append the new element to 3. Add the element to the DOM tree by appending it to the parent element
